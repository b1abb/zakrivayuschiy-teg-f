# Закрывающий тег

Проектная работа по адаптивной вёрстке и CSS-анимациям. Страница собрана как дневник прохождения учебных спринтов: карточки с изображениями, подписями, текстом воспоминаний, анимированной кнопкой лайка и модальным окном сохранения.

Репозиторий: https://github.com/b1abb/zakrivayuschiy-teg-f
GitHubPages: https://b1abb.github.io/zakrivayuschiy-teg-f/

## Функциональность

- Адаптивная вёрстка для десктопных и мобильных экранов.
- Восемь карточек с изображениями и CSS-фильтрами.
- Анимированная SVG-иконка сердца для лайка.
- Переключение текста кнопки `Like` / `Unlike`.
- Модальное окно на элементе `dialog`.
- Фиксированный пиксельный фон на CSS-градиентах.

## Технологии

- HTML5
- CSS3
- JavaScript
- БЭМ

## Структура проекта

```text
.
├── fonts/       # локальные шрифты
├── images/      # изображения карточек и favicon
├── scripts/     # скрипты лайка и модального окна
├── styles/      # глобальные стили, переменные, основные стили и анимации
├── svg/         # SVG-иконки
└── index.html   # главная страница
```

## Запуск

Проект не требует сборки. Откройте `index.html` в браузере
